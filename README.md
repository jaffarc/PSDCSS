# Installation

	Either through forking or by using [npm](http://npmjs.org) (the recommended way):

    npm install -g psdcss
# Usage
	
	project/www $ psdcss


## Pattern  

		
	Project	
		|
		|.gitignore
		|psd.json
		|package.json
		|README.md
